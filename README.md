# Финикон — Учёт личных финансов

Дипломная работа / Мурнаев Дмитрий Юрьевич

---

## О проекте

Это моё приложение для учёта личных доходов и расходов. Идея появилась потому, что я хотел понять, куда уходят деньги и научиться планировать бюджет. Сначала делал простую таблицу в Excel, но потом решил сделать полноценное веб-приложение.

На данный момент приложение умеет:
- Добавлять доходы и расходы
- Переводить деньги между счетами
- Создавать цели и откладывать на них деньги
- Смотреть статистику по месяцам
- Работать как PWA на телефоне (можно установить на рабочий стол)

---

## Стек технологий

Для разработки я выбрал:
- **Frontend**: React + Vite — moderneй фреймворк, который я изучаю
- **Styling**: Tailwind CSS — удобно делать адаптивный дизайн
- **Backend**: Node.js + Express — простой сервер на JavaScript
- **Database**: PostgreSQL — надёжная база данных
- **Deploy**: Render — бесплатный хостинг для веб-приложений

Также использовал:
- JWT для авторизации
- bcrypt для хэширования паролей
- Axios для запросов к API

## Функционал

### Аккаунты пользователя
- Регистрация и вход по email и паролю
- Смена пароля
- Выбор валюты (рубли, доллары, евро)

### Счета
- Создание нескольких счетов (наличные, карта, вклад)
- Каждый счёт со своим балансом
- Редактирование и удаление счетов

### Транзакции
- Добавление расходов и доходов
- Выбор категории и счёта
- Установка даты
- Редактирование и удаление
- Фильтрация по месяцам

### Переводы
- Перевод денег между своими счетами
- Автоматическое изменение балансов

### Цели
- Создание целей (например "на машину" или "на отпуск")
- Внесение денег в цель
- Отслеживание прогресса (сколько накоплено из нужной суммы)
- Удаление целей

### Статистика
- Общая сумма доходов и расходов
- Диаграмма распределения по категориям

### Интерфейс
- Адаптивный дизайн для телефона и компьютера
- Тёмная тема
- PWA — можно установить и пользоваться офлайн частично
- Удобное мобильное меню снизу экрана

## Как запустить у себя

### Требования
- Node.js 18+
- PostgreSQL (локально или в облаке)

### Установка

1. Склонировать репозиторий:
```bash
git clone https://github.com/DmitryMurnaev/finance-tracker.git
cd finance-tracker
```

2. Установить зависимости:
```bash
cd server && npm install
cd ../client && npm install
```

3. Настроить переменные окружения:

В `server/.env`:
```
PORT=5000
DB_HOST=localhost
DB_PORT=5432
DB_NAME=finance_tracker
DB_USER=postgres
DB_PASSWORD=твой_пароль
JWT_SECRET=любая_секретная_строка
```

В `client/.env`:
```
VITE_API_URL=http://localhost:5000/api
```

4. Запустить:
```bash
# В одном терминале
cd server && npm run dev

# В другом терминале
cd client && npm run dev
```

Приложение откроется на http://localhost:5173
